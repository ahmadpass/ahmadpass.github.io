Flutter 101: A Rookie Guide

Install & Setup

Posted by Ahmad R.A on December 09, 2022

Maybe you have been wondering about how fast the software industries grows up lately in the last decade. At the time now, theres many UI framework on sight for running beautiful App that can run on multiple platform. And yes, Flutter is one of them!

This article will help you know about how to install and setup flutter for the first time, along with some practical examples.

Preparation

Before anything else, we should prepared for what will come next. On minimal requirement based on Flutter docs, you should prepare:

  • Windows 10 or later (64-bit), x86-64 based operating systems
  • Disk space at least 1.64 GB (does not include disk space for IDE/tools)
  • Some tools flutter depends on, like Windows PowerShell 5.0 or newer (this is pre-installed with Windows 10) and Git for Windows 2.x, with the Use Git from the Windows Command Prompt option

Now, after our PC/laptop is well prepared, it's time to do the installation process!

Step 1: Flutter SDK Download

As the first step, we should install the SDK on Flutter official website here: Flutter: Get SDK

You should see this screen, now choose the right operating system you are using

Next, download the SDK. I will demonstrate using Windows for the next process of installation. If you are using the different OS, don't worries, theres an instruction on documentation that will help you. Just click on your prefered OS.

Step 2: Add Flutter into The Path

After the download finished, in order for us to be able using Flutter command on regular windows console (such as command prompt), we need to add Flutter to the path environtment variable. First, navigate to your downloaded zip file and extract the zip file to your preffered location. In my case, I create a new folder 'tooling' inside a folder 'dev' in local disk C and put the extraction folder there.

Here I change the extraction folder name to 'flutter'

Then, navigate to the 'bin' folder and copy the path into that folder.

Look at the hightlighted path, copy that

Next, click Windows and search for 'Edit the system environment variables', open it and then click 'Environtment Variables...' on the bottom right corner.

Click the blue hightlighted ones

After that go for 'User variable for [Your User Name]' and click on 'Path'

Click the blue hightlighted ones

Click on 'New' button, then paste your bin path

Paste it here

Now, do the same step for the .pub-cache file

Just like the step before

Done, finishing it by click the 'OK' button.

Step 3: Check The Diagnostic, Flutter Doctor!

After successfully added Flutter into the path, we should check if there is any platform dependencies Flutter needed you haven't install yet. First, open your Command Prompt. If you already have opened the Command Prompt before the Step 2, you need to close that Command Prompt and open a new one in order for the changes we made at Step 2 took place. Next, run the following command:

                            flutter doctor

Then enter and see how it run. The output should be like this:

If you have already install all dependencies, it should finished with no issues.

If you found any issues, the Flutter Doctor will show you the detail about your issues. That will give you a clue how to solve that. But most of it because you haven't installed the dependencies needed. Here are the list of dependencies we are needed:

  • Chrome Browser
  • Visual Studio
  • Android Studio
  • Visual Studio Code

In case you are never download SDK Manager before, you will got this issue: 

                            [X] Android toolchain - develop for Android devices
                                X Unable to locate Android SDK.
                                Install Android Studio from:https://developer.android.com/studio/index.html
                                On first launch it will assist you in installing the Android SDK components.
                                (or visit https://flutter.dev/docs/get-started/install/windows#android-setup 
                                for detailed instructions).If the Android SDK has been installed to a custom
                                location, please use `flutter config --android-sdk` to update to that location.

What you have to do is:

  1. Install Java JDK
  2. Install SDK Manager

The alternative solution for this issue, you can simply install SDK Manager using Android Studio you have downloaded before. First, open your Android Studio, click 'More Actions', and select 'SDK Manager' on dropdown options.

Here is the start page of the Android Studio

Now, navigate into 'SDK Tools' tab, and checked the neccessary tools for install. Then, click 'Apply' and 'OK' for implement the changes. Click 'OK' on confirmation Pop Up.

Checked the hightlighted ones

Now, try to run again the diagnostic using additional command to accept the licenses:

                            Flutter doctor --android-licenses

This will run the doctor and also accepted the SDK licenses

And now you should be done with the installation process. Happy coding :)
